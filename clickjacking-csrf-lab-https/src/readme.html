Clickjacking + CSRF token (HTTPS)

  1. Start: docker compose up -d
  2. Login at https://10.10.0.2:8443/login.php (accept the self-signed cert), user alice / alice123.
  3. Host a decoy page on http://127.0.0.1:8000 that frames https://10.10.0.2:8443/account.php.
  4. The session cookie is set with SameSite=None; Secure, so third-party iframes can send it. Clicking the decoy should delete the account.
